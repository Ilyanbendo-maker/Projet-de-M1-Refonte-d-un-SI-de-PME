# Projet-de-M1-Refonte-d-un-SI-de-PME
une PME victime d'un grave incident de sécurité a perdu l'intégralité de son SI et nous mandate, en tant qu'ingénieurs SSII, pour le reconstruire intégralement. Contraintes : solutions libres et gratuites en priorité, versions stables, montée en charge jusqu'à 1000 utilisateurs simultanés, conformité légale.
